# Starter Workflow Example

This is one possible workflow for the starter app. Students may redesign it.

## Possible Project State

- topic or research idea
- problem and motivation
- novelty or prior-work gap
- method or agent workflow
- evaluation plan
- milestones, risks, and resources
- references, assumptions, or source notes

## Possible Stages

- collect a rough idea;
- ask or suggest clarifying details;
- update project state from student feedback;
- draft proposal content;
- build a compliance matrix;
- produce an evaluation report;
- revise weak sections.
