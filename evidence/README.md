# Workflow evidence

This folder holds artifacts that show the agent was used to **draft, critique, and revise** proposal content—not just to paste a finished PDF.

## What to include when submitting

- **Screenshots** of the app during literature search, agent Q&A, proposal generation, compliance matrix, and evaluation report.
- **Export logs** or saved `transcript` objects from API responses (redact API keys).
- **Before/after** snippets when a field was refined or a weak section was revised.
